# 基于RT-Thread的智能桌面小工具

## 1.作品描述

本项目是基于国民芯片N32G457，和RT-thread操作系统开发的智能桌面小工具。由于万物互联深入人心，于是便有了把自己的家庭也做成一个小的互联型家庭的想法。此项目主要是通过NTP获取网络时间校对，并使用软件RTC计时，将时间在OLED上显示出来；还可以通过温湿度模块把环境的温湿度采集出来，房间的空气质量采集出来并在OLED上显示；还有获取当地实时的天气状况的功能。作品的整体样貌如下：
